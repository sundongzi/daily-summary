- `Array.with`：用于改变数组中的某一项且不影响原数组

  ```javascript
  const numbers = [1, 2, 3, 4, 5];
  const updatedNumbers = numbers.with(1, 'new value');
  console.log(updatedNumbers);             // [1, 'new value', 3, 4, 5]
  console.log(numbers === updatedNumbers);  // false
  ```
